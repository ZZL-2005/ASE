## 当前问题

### 1. Rocket.Chat API 限流 (429 错误)
**现象：** Agent 轮询消息时频繁收到 429 错误
**原因：** 每 2 秒轮询一次，触发 Rocket.Chat 的速率限制
**影响：** 无法稳定接收新消息

**解决方案：**
- 增加轮询间隔（5-10秒）
- 使用 WebSocket 替代轮询
- 实现指数退避重试

### 2. 消息发送成功但不稳定
**现象：** 有时能发送，有时不能
**原因：** API 限流导致不稳定
**状态：** 已修复 outbound 消息处理逻辑
